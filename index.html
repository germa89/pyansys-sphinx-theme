PyAnsys Sphinx Theme¶

Introduction and Purpose¶

The PyAnsys documentation theme is an extension of the popular PyData Sphinx Theme used by numpy, pandas, PyVista and a variety other packages. This theme was packaged a consistent documentation pattern and “feel” across the all PyAnsys packages.

Documentation¶

Full documentation for the theme can found at PyAnsys Sphinx Documentation. The webpage was also built using the pyansys-sphinx-theme, so visit the site for a preview of the theme.

Other Sites Using the PyAnsys theme:

  • PyMAPDL

  • DPF-Core

  • DPF-Post

  • Legacy PyMAPDL Reader

Getting Started¶

Install this theme with:

pip install pyansys-sphinx-theme

Next, modify your sphinx conf.py to use html_theme = 'pyansys_sphinx_theme'. If you are just getting started using sphinx, follow the directions at Sphinx Quickstart.

For further details, visit PyAnsys Sphinx Theme Usage
